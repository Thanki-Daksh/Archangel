# 🗺️ Roadmap

> Defines the long-term development plan for The Archangel.

---

# Purpose

This document exists to record the future direction of The Archangel.

It is intentionally separate from the implementation specification so that long-term goals can evolve without affecting the architecture documents.

---

# Scope

This document may contain:

- Major versions
- Planned features
- Milestones
- Future ideas
- Long-term objectives
- Vision beyond v1

There is intentionally no predefined format.

The roadmap should be updated whenever the project's priorities or long-term direction changes.

---

# Note

Unlike the other documents in `/docs`, this file is expected to change frequently throughout the project's lifetime.

It serves as the project's planning document rather than a technical specification.
